Abstract

ARTZY it is a website for artists in all fields. The site aims to unite the communities of artists and expand their horizons. Artzy is the future social network for the artist, they will be able to upload their works of art and share them with the rest of the community. At the same time, they will be able to find new passions.

1.Introduction

1.1 Report Details

This document represents the ARTZY report. The document is encoded in UTF-8 and transmitted with a text / html media type. The project is realized in adobe xd.

2.Overview

2.1.Website features

The website conteins:

  • Main Page "Home";
  • "Sing up/in" Pages;
  • "Profile" Page;
  • "Profile settings" Page;
  • "Uploade" Page;
  • "Chat" Page;
  • "Draw with friends" Page;

2.2 Operating environment

The websitw is accessible to any device that has internet connection and a web browser.

2.3 User documentation

The first page the user accesses is the "Home" page where he will see the newest post on the platform.

3.Description Interface

3.1 Design

  • Main Page : "Home"

    • The main page is divided into 3 areas, a box on the left that contains the site logo, a button for settings and a button for the chat page. At the top we have a box containing the search bar, the most popular hashtags, the registration button, the upload button and the profile picture of the user who acts as a button to the profile. And in the middle of the page is the main area where users can scroll to see the new posts.

  • "Sing up/in" Pages

    • On this page you will find a form for registration or connection to the site. To choose only the desired option, click on the box that says "Sing in" or "Sing up" and then you will defend the desired form.

  • "Profile" Page

    • On the profile page the user can see how many followers, follow-ups and likes he has. Below the page you can see the posts made, and by accessing the "save" button you can see the posts saved in albums.

  • "Profile settings" Page

    • In the settings page you can change the username and password of the account and upload the profile picture.

  • Uploade" Page

    • There is a upload form for the posts, each post consists of a picture with the title and description in the description you can add hashtags.

  • "Chat" Page

    • It contains a box that represents the chat and a box where you can write and send by pressing the send button. On the right is a box in which users who participate in the conversation will appear. On this page on the left appears a special pencil-shaped button next to the "Draw with friends" page.

  • "Draw with friends" Page

    • This page is intended to amuse users, they will have a pencil with which they can draw with the rest of the connected users.
